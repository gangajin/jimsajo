## 📌 프로젝트 이름
jimsajo - 동남아 여행 패키지 구매 사이트

## 1️⃣ 프로젝트 개요

### 📚 소개
여행을 떠나고 싶지만 당장 갈 수 없는 상황에서, 팀원들과 함께 여행 가는 설렘을 느끼고 싶었습니다. 그래서 동남아 여행 패키지를 함께 구매하고, 여행 계획을 이야기하며 출발 전의 설렘을 나눌 수 있는 웹사이트를 만들게 되었습니다.
